# An end-to-end platform for first-time founders that takes them from raw idea ...

Static landing page — no build step required.

## Running locally

```bash
# Option 1: open directly in browser
open index.html

# Option 2: serve with any static server
npx serve .
# or
python3 -m http.server 8080
```

Then visit http://localhost:8080 (or http://localhost:3000 for `npx serve`).

## Notes

- Tailwind CSS loaded via CDN — no build step needed.
- Fonts loaded from Google Fonts (Space Grotesk + Inter).
- Place your logo asset at `public/logo.png`.
- All external requests use HTTPS.
